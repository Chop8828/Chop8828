## Projects

### Automatic Plant Irrigation System
Designed an Arduino-based irrigation system that monitored soil moisture, light, temperature, and humidity to automatically water plants. The system included LCD feedback, Excel data logging, and water pump control.

### Engineering Class Scheduler
Developed a Python and Tkinter-based scheduling application that imports course data, detects time conflicts, visualizes weekly schedules, and exports finalized schedules using CSV files.

### Audio Amplifier PCB
Designed and implemented a custom PCB in Eagle for a low-noise headphone amplifier based on a provided schematic. Generated Gerber files, optimized component placement, and soldered through-hole components.

### 16×8 SRAM Design in 16nm CMOS
Designed transistor-level SRAM cells and pre-charge circuitry using Synopsys Custom Compiler. Performed layout optimization, DRC/LVS verification, and performance evaluation.

### ASIC Motion Estimator in 14nm CMOS
Implemented a motion estimator for video compression using Verilog RTL and completed ASIC flow through synthesis, physical design, and timing sign-off using Synopsys tools.

### Enhanced Gait Analysis Wearable Technology
Developed a wearable gait analysis system using EMG, FSR, and kinematic sensors to detect heel strike, mid-stance, and toe-off in real time with Python visualization.

### Binary to Number System Converter
Created a dual-core STM32MP157 embedded system where the Cortex-M4 handled GPIO binary inputs and the Cortex-A7 converted values into decimal, octal, and hexadecimal using OpenAMP/RPMsg.

### Auditory Emotion Identification Using Machine Learning
Built a machine learning pipeline to classify emotions from speech audio using MFCCs, spectral features, and a tuned quadratic SVM model in MATLAB.
